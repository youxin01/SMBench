随着现代科学技术的发展,每年都有大量的学术论文发表。如何衡量学术论文的重要性,成为学术界和科技部门普遍关心的一个问题。有一种确定学术论文重要性的方法是考虑论文被引用的状况,包括被引用的次数以及引用论文的重要性程度。假如我们用邻接矩阵来表示论文引用关系,。现有 A、B、C、D、E、F共6篇学术论文,它们的引用关系下面的矩阵所示。设计依据上述引用关系排出6篇论文重要性顺序的模型与算法,并给出用该算法排得的结果。
引用关系：
|      | A    | B    | C    | D    | E    | F    |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| A    | 0    | 1    | 0    | 0    | 0    | 0    |
| B    | 0    | 0    | 1    | 0    | 0    | 0    |
| C    | 1    | 0    | 0    | 1    | 0    | 0    |
| D    | 1    | 0    | 0    | 0    | 1    | 1    |
| E    | 0    | 0    | 1    | 0    | 0    | 1    |
| F    | 0    | 1    | 0    | 0    | 0    | 0    |
